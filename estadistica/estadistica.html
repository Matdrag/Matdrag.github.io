MEDIDAS DE TENDENCIA CENTRAL
MEDIA

La media, también conocida como promedio, es una medida de tendencia central que se utiliza para representar el valor típico o central de un conjunto de datos numéricos. Se define como la suma de todos los valores en el conjunto de datos dividido por el número total de valores en el conjunto. La fórmula para calcular la media es:
media = (x1 + x2 + ... + xn) / n

Donde:

  • x1, x2, ..., xn son los valores que se suman en el conjunto de datos.
  • n es el número total de valores que se suman en el conjunto de datos.

    • ejemplo 1:
    Si tenemos un conjunto de datos de 5 números: 2, 4, 6, 8, 10, podemos calcular la media utilizando la fórmula de la siguiente manera:
    media = (2 + 4 + 6 + 8 + 10) / 5 media = 30 / 5 media = 6
    Por lo tanto, la media de este conjunto de datos es 6.

    ejemplo 1:
    Supongamos que tenemos los siguientes números: 3.5, 4.2, 5.1, 2.8, 6.3.Para calcular el promedio, primero sumamos todos los números:

    3.5 + 4.2 + 5.1 + 2.8 + 6.3 = 21.9

    Luego, dividimos la suma por el número total de números, que en este caso es 5:

    21.9 / 5 = 4.38

    Por lo tanto, la media o promedio de este conjunto de números es 4.38.

    Recuerda que la media es una medida de tendencia central que puede ser útil para resumir conjuntos de datos numéricos, pero también es importante tener en cuenta otras medidas estadísticas como la mediana y la desviación estándar para obtener una imagen más completa de los datos.
